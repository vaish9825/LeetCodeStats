# LeetCodeStats Dashboard

A web-based dashboard that fetches and visualizes real-time LeetCode profile statistics using the LeetCode GraphQL API.

## Features

* Search any LeetCode username
* Real-time coding statistics and submission metrics
* Animated circular progress bars for Easy, Medium, and Hard problems
* Dynamic stat cards with responsive UI
* API integration using GraphQL and async/await
* Smooth and clean user experience

## Tech Stack

* HTML5
* CSS3
* JavaScript
* GraphQL API

## How It Works

The user enters a LeetCode username, and the app sends a GraphQL POST request to fetch profile statistics. The response data is then processed and displayed dynamically through progress indicators and metric cards.
